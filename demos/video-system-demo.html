JavaScript API

Open Video Player

// Open video for a player openVideoPlayer('Fernando Mendoza'); // Open specific video openVideoPlayer('Fernando Mendoza', 'abc123xyz');

Get Player Videos

// Get all video data for a player const videos = getPlayerVideos('Fernando Mendoza'); // Returns: { mainHighlight, playlist, stats, position, school }

Create Playlists

// Create playlist by position const qbPlaylist = createVideoPlaylist('QB'); const wrPlaylist = createVideoPlaylist('WR'); // Create top prospects playlist const topProspects = createVideoPlaylist('Top Prospects');

Get Recommendations

// Get related videos after watching const recommendations = getRecommendedVideos('Fernando Mendoza', 3); // Returns array of similar position players

Video Thumbnails

// Get thumbnail URL const thumb = getVideoThumbnail('abc123xyz', 'hqdefault'); // Qualities: default, mqdefault, hqdefault, maxresdefault

Integration Guide

Include CSS and JavaScript

Add the video system files to your HTML head and body:

<!-- In <head> --> <link rel="stylesheet" href="css/youtube-videos.css"> <!-- Before </body> --> <script src="js/youtube-videos.js"></script>

Add Video Data

Video data is already included in the youtube-videos.js file. To add more players, extend the playerVideos object:

const playerVideos = { "Player Name": { mainHighlight: "youtube-video-id", playlist: [ { id: "video1", title: "Highlights", duration: "8:42" }, { id: "video2", title: "vs Team", duration: "4:15" } ], stats: { views: "1.2M", likes: "45K", channel: "Channel" }, position: "QB", school: "School" } };

Player Cards

Ensure player cards have the data-player attribute:

<div class="pick-card" data-player="Fernando Mendoza"> <!-- Video indicator will be auto-added --> </div>

Big Board Integration

For Big Board table, add video column to your render function:

// In renderBigBoard(): const hasVideo = playerVideos[player.name] ? `<button class="bb-video-btn" onclick="openVideoPlayer('${player.name}')"> <i class="fas fa-play"></i> </button>` : '';

Data Structure

const playerVideos = { "Player Name": { mainHighlight: "youtube-video-id", // Primary highlight video playlist: [ // Array of all videos { id: "video-id", // YouTube video ID title: "Video Title", // Display title duration: "8:42", // Video duration thumbnail: "mqdefault" // Thumbnail quality } ], stats: { views: "1.2M", // Total/avg views likes: "45K", // Total/avg likes channel: "Channel Name", // YouTube channel videoCount: 4 // Number of videos }, position: "QB", // Player position school: "School Name" // College } };
