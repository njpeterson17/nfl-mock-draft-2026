Usage Examples

Basic HTML Usage

<!-- With data attributes (auto-loads) --> <div class="player-photo" data-player-name="Fernando Mendoza" data-player-position="QB" data-player-school="Indiana"> </div> <!-- JavaScript loading --> <div id="playerPhoto" class="player-photo"></div> <script> loadPlayerImage('playerPhoto', 'Fernando Mendoza', 'QB', 'Indiana'); </script>

JavaScript API

// Create player photo element const photo = createPlayerPhoto({ name: 'Caleb Downs', position: 'S', school: 'Ohio State' }, { size: 'large' }); // Load into existing container loadPlayerImage(container, 'Arvell Reese', 'LB', 'Ohio State'); // Get image URL const url = await playerImageLoader.loadPlayerImage(name, position, school); // Refresh all images refreshPlayerImages();

React/Vue Component Props

const props = getPlayerPhotoProps({ name: 'Jeremiyah Love', position: 'RB', school: 'Notre Dame' }); // Returns: { src: 'images/players/jeremiyah-love.jpg', alt: 'Jeremiyah Love', fallback: '...' }
