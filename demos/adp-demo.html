Integration

The ADP system is automatically integrated into the main NFL Mock Draft site. Here's how it works:

  1. Navigation Tab: Click the "ADP" tab in the main navigation to view the ADP Leaderboard
  2. Pick Cards: ADP badges appear on pick cards showing value/reach/fair status
  3. Big Board: ADP column is added to the Big Board view
  4. Toggle: Use the "Show ADP" checkbox in the controls bar to toggle visibility
  5. Tooltips: Click any ADP badge to see detailed information
