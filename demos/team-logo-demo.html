üíª Usage

Basic Usage

// Get logo HTML with wrapper const logoHtml = TeamLogoSystem.getLogo('chiefs', 64); // Get raw SVG const svgContent = TeamLogoSystem.getLogoSvg('chiefs'); // Get team colors const colors = TeamLogoSystem.getColors('chiefs'); // { primary: '#E31837', secondary: '#FFB81C', text: '#FFFFFF', accent: '#FFFFFF' } // Check if logo exists const hasLogo = TeamLogoSystem.hasLogo('chiefs'); // true

Auto-initialization

<!-- Add data attributes to HTML --> <div data-team="chiefs" data-size="64"></div> <!-- Logos auto-initialize on DOM ready --> // Or manually trigger: TeamLogoSystem.initialize(document);

Name Normalization

// All of these return 'chiefs' TeamLogoSystem.normalizeName('KC'); TeamLogoSystem.normalizeName('kansas city'); TeamLogoSystem.normalizeName('Kansas City Chiefs'); TeamLogoSystem.normalizeName('chiefs');
