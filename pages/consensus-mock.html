How Consensus is Calculated

Weighted Voting System

Each source contributes to the consensus based on their assigned weight. PFF serves as the primary source with 35% weight.

PFF 35%
ESPN 25%
PFN 20%
TDN 15%
WF 5%

Confidence Formula

Confidence = (Weighted Votes for Consensus Pick) / (Total Weighted Votes)

For example, if PFF (35%) and ESPN (25%) both pick Player A with 90% confidence:

  • Weighted votes for Player A = (0.35 √ó 0.90) + (0.25 √ó 0.90) = 0.54
  • Total weighted votes = 0.35 + 0.25 = 0.60
  • Confidence = 0.54 / 0.60 = 90%
